# 指标口径与停机条件

## 目标口径（按月）

- **收益口径**：基于模拟的单月收益率分布，按每条路径的月度 PnL / 资金基准（capital_base）计算。
- **核心目标**：月度收益分布的中位数 ≥ 1%。
- **风控约束**：
  - 亏损概率 ≤ 25%。
  - 左尾风险：月度收益 5% 分位数 ≥ -10%。

## 收敛与预算停机

- **最小迭代轮数**：至少 3 轮后才允许因达标停机，避免过早停止。
- **收敛停机**：连续 3 轮中位数收益提升 < 0.1%（绝对值）。
- **预算停机**：最大迭代轮数 8。

> 注：上述阈值与 `sim/loop.py` 中的默认配置一致，确保迭代与报告可复现。
