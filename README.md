# 红枣期权+期货对冲：LLM闭环仿真（最小版）

你给的数据：
- `红枣期货数据.CSV`：日频期货 OHLC + 成交量
- `红枣期权数据.CSV`：日频隐含波动率汇总（均值/最低/最高/标准差/合约数量/成交量）

由于缺少“期权链明细”（到期/行权价/C-P/期权价格/Greeks），本项目采用**合成期权**近似：用当日均值 IV（可选用 min/max/std 做压力情景）+ Black-Scholes 生成“卖Put/卖Call”的价格与Delta，并用期货做动态对冲，保留 10% 敞口。

## 运行
1) 安装依赖

```bash
pip install -r requirements.txt
```

2) 运行基线仿真（会输出月度收益分布、最大回撤分布、亏损概率等）

```bash
python -m sim.run
```

## 重要说明（模型近似）
- **这不是**真实“按具体合约链双卖”的回测；结果用于验证思路与风险形态。
- 若补齐期权链明细，可把合成期权替换为真实合约级定价/结算。
